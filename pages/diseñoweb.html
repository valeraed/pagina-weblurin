¿Te interesa el mundo del Diseño y Programación Web?

¡La carrera de Diseño es ideal para tí!

La carrera de Diseño y Programación Web te permitirá adquirir conocimientos y habilidades para la creación y gestión de sitios web dinámicos, con un enfoque en diseño gráfico, experiencia de usuario (UX), y desarrollo frontend y backend.

CONOCER MÁS...

Conoce la Carrera

El profesional en Diseño y Programación Web está capacitado para crear sitios funcionales, atractivos y accesibles, aplicando principios de diseño, usabilidad y optimización para la web. Aprenderás a desarrollar proyectos digitales de manera integral, empleando herramientas y técnicas actuales para ofrecer soluciones innovadoras en un entorno tecnológico en constante cambio.

¿Qué aprenderás?

1. Diseño de interfaces web atractivas y funcionales.

2. Programación con lenguajes como HTML, CSS y JavaScript.

3. Desarrollo de aplicaciones web utilizando frameworks y tecnologías modernas.

Certificaciones

Al culminar tu carrera, podrás obtener las siguientes certificaciones:

• Diseño y desarrollo de sitios web

• Implementación de sistemas web

• Gestión de proyectos de desarrollo web
